# Security Policy

## Supported Versions
The table below shows which versions of this project currently receive security updates.

| Version | Supported |
|--------|-----------|
| 1.x.x  | ✔️ Active |
| 0.x.x  | ❌ No longer supported |

## Reporting a Vulnerability
If you discover a security vulnerability, please report it responsibly.

### 📩 How to report
Send an email to: **alae.security.report@gmail.com**  
(You can replace this with any email you prefer.)

### 🕒 Response Time
You can expect:
- An acknowledgment within **48 hours**
- A first update within **5–7 days**
- Continuous updates until the issue is resolved

### 🔒 Responsible Disclosure
Please:
- Do **not** publicly disclose the vulnerability before it is fixed  
- Provide as much detail as possible (steps to reproduce, screenshots, etc.)

If the vulnerability is valid, I will:
- Work on a fix as soon as possible  
- Credit you (optional) in the changelog or release notes  

Thank you for helping keep this project secure.
